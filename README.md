## 📝 소개

이 레포지토리는 매일의 개발 학습 내용과 작업한 코드를 기록하는 개인 일지입니다. 개발 과정에서 배운 내용, 해결한 문제, 새로운 인사이트를 꾸준히 기록하여 성장의 여정을 남기고자 합니다.

## 🗂️ 구조

일일 학습 기록은 다음과 같은 형식으로 작성됩니다:

```markdown
# YYYY-MM-DD (요일)

## 📚 [알고리즘]

- 문제 해결 내용
- 학습한 알고리즘 개념

## 🛠️ [백엔드/프론트엔드/기타 분야]

- 특정 기술 학습 내용
- 프레임워크/라이브러리 사용법

## 🔍 [오늘의 문제/해결]

- 마주친 문제와 해결 방법

## 📌 [참고 자료]

- 도움이 된 리소스 링크

## 📝 [내일 할 일]

- 다음 날 계획

## 💡 [배운 점/인사이트]

- 기술적 인사이트
- 개발 관련 깨달음

## 🤔 [느낀 점]

- 학습 과정에서의 소감
- 개인적인 성찰

## ⏱️ [학습/작업 시간]

- 분야별 투자 시간
- 총 학습시간
```
